# 🧺 Detergent Brand Perceptual Map

## 📌 Overview
This project presents an interactive perceptual map of detergent brands in India based on consumer behavior analysis. It visualizes how brands are positioned in the minds of consumers across key attributes like price and quality.

## 🎯 Objective
- Understand consumer perception of detergent brands  
- Analyze competitive positioning  
- Identify gaps and opportunities in the market  

## 🏷️ Brands Included
- Surf Excel  
- Ariel  
- Tide  
- Ghadi  
- Other  

## 📊 Methodology
- Data collected through a structured survey  
- Responses analyzed to identify key perception dimensions  
- Perceptual map created using data visualization techniques  

## 🛠️ Tools Used
- HTML  
- Basic data visualization
